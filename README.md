## Usage

1. Clone project to your local computer:

```
$ git clone https://github.com/pipipi666/middle.messenger.praktikum.yandex.git
```

2. Install the packages:

```
$ npm install
```

3. Run the app:

```
$ npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
5. Build the app for production to the `dist` folder and start the project:

```
$ npm run start
```
